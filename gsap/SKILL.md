# GSAP Skills

GSAP (GreenSock Animation Platform) 动画库的完整技能集。通过公共 npm 包安装：`npm install gsap`，无需私有 registry 或 auth token。

## Sub-Skills

### gsap-core

Core API：`gsap.to()`、`from()`、`fromTo()`、easing、duration、stagger、defaults、transforms、autoAlpha、`gsap.matchMedia()`（响应式、prefers-reduced-motion）。

触发词：animation library, JavaScript animation, JS animation, React animation, Vue animation, GSAP tweens, easing, stagger, basic animation, transform, opacity, responsive, accessibility, reduced motion, matchMedia.

### gsap-timeline

Timelines：`gsap.timeline()`、position parameter、labels、nesting、playback control。

触发词：sequencing, timeline, keyframes, choreograph, multi-step animation, animation order.

### gsap-scrolltrigger

ScrollTrigger：scroll-linked animations、pinning、scrub、triggers、refresh、cleanup。

触发词：scroll animation, scroll-driven animation, parallax, pin section, ScrollTrigger, pin, scrub.

### gsap-plugins

Plugins：ScrollToPlugin、ScrollSmoother、Flip、Draggable、Inertia、Observer、SplitText、ScrambleText、SVG/physics plugins、CustomEase、EasePack、GSDevTools。所有插件免费，无需 Club GSAP 会员。

触发词：plugin, scroll-to, flip, draggable, MorphSVG, DrawSVG, MotionPath, SplitText, ScrambleText, CustomEase, registerPlugin, Club GSAP, GSAP license, bonus plugins.

### gsap-utils

`gsap.utils`：clamp、mapRange、normalize、interpolate、random、snap、toArray、wrap、pipe。

触发词：gsap.utils, clamp, mapRange, random, snap, toArray, wrap, interpolation.

### gsap-react

React integration：useGSAP hook、refs、`gsap.context()`、cleanup、SSR。

触发词：React animation, Next.js animation, useGSAP, cleanup on unmount, GSAP React.

### gsap-performance

Performance：transforms、will-change、batching、ScrollTrigger optimization。

触发词：performance, 60fps, jank, animation performance, optimize.

### gsap-frameworks

Vue、Svelte 及其他框架：lifecycle、scoping selectors、cleanup on unmount。React 请使用 gsap-react。

触发词：Vue, Svelte, Nuxt, SvelteKit, framework, lifecycle, onMounted, onUnmounted.
