# 术语表

## UI 组件库

一组可复用界面组件、样式规则和交互模式的集合。它可以表现为运行时依赖，也可以表现为源码分发、registry 或可复制代码片段。

## 源码分发

组件代码通过 CLI、registry 或复制方式进入项目代码库，团队拥有后续修改和维护责任。shadcn/ui、ReUI、shadcnblocks、Kokonut UI、Tailark UI、UI TripleD 均属于此类或与此类高度相关。

## 运行时依赖

组件通过 npm 包在运行时导入，升级、主题能力和行为由上游包控制。HeroUI 与 Material UI 属于此类。

## 展示页面

用于表达单个 UI 组件库视觉语言、适用场景和组件组织方式的 React 页面。展示页面不等同于官方组件复制品，应采用原创布局表达该库风格特征。

## 组件库元数据

统一描述组件库名称、定位、技术基础、适合场景、注意事项与来源链接的数据结构。展示项目通过该数据结构生成导航和页面基础内容。
